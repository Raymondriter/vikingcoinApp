# vikingcoinApp

An Android application that provides a WebView wrapper for [vikingcoin.org](https://vikingcoin.org).

## Tech Stack

- **Platform:** Android
- **Language:** Kotlin / Java
- **Build Tool:** Gradle

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (Arctic Fox or later)
- Android SDK (API level 21+)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Raymondriter/vikingcoinApp.git
   ```
2. Open the project in Android Studio.
3. Sync Gradle dependencies.
4. Run the app on an emulator or physical device.
